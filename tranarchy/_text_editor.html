Design.

A text editor - even one as simple as ED: THE STANDARD EDITOR - never provides a direct interface to the underlying text. It shows an interpretation of the text, which is a prerequisite for interaction. "Text" in computers is not really text. It is a series of bytes. A text editor cuts certain corners in order to show you this series of bytes in a way that pretends it is text.
